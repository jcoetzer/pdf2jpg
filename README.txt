Install CutePDF:
----------------

Convert to PDF documents on the fly — Free Software
 
	
Portable Document Format (PDF) is the de facto standard for the secure and reliable distribution and exchange of electronic documents and forms 
around the world.  CutePDF Writer is the free version of commercial PDF converter software. CutePDF Writer installs itself as a "printer subsystem". 
This makes it possible for virtually any Windows applications (must be able to print) to create PDF documents.

http://www.cutepdf.com/download/CuteWriter.exe

Install ImageMagick and Ghostscript:
------------------------------------

Download a prebuilt binary of ImageMagick for your architecture (win32 or win64) from the following link:

http://www.imagemagick.org/download/binaries/

Ghostscript is available at:

https://www.ghostscript.com/download/gsdnld.html
